# result predictor

This is a simple Python project that reads student marks from a CSV file and shows Pass/Fail status along with division.

## Features
- Read data from CSV file
- Show Pass/Fail
- Assign Division (First, Second, Third)

## CSV Format
Name,Maths,Science,English
Rahul,75,68,80
Priya,40,35,30
rohit,65,45,50

## How to Run
python result.py

## Output Example
Rahul → Pass (First Division)
Priya → Fail
rohit → pass(Second Division)
